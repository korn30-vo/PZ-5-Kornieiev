# Лабораторна робота 5
# Виконав студент групи ПДМ-51 Корнєєв Володимир
Ця лабораторна робота присвячена реалізації згорткової нейронної мережі (CNN) для задачі класифікації зображень із використанням набору даних Fashion-MNIST на мові програмування Python.

Метою роботи є ознайомлення з основами глибокого навчання, попередньою обробкою зображень, побудовою та навчанням нейронної мережі, а також оцінкою її ефективності.

У процесі виконання лабораторної роботи виконується завантаження датасету через бібліотеку Keras, аналіз його структури та візуалізація прикладів зображень. Дані проходять етапи попередньої обробки: зміна форми, нормалізація пікселів та перетворення міток у one-hot encoding. Після цього набір даних розділяється на навчальну та тестову вибірки.
